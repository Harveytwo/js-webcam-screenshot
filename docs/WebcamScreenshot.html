goCallback(code, result)

Callback called after the go() method ends.
Parameters:
Name Type Argument Description
code WebcamScreenshot.RC One of the WebcamScreenshot.RC values.
result string | Object <optional>
 This value may be:
  • If code is not WebcamScreenshot.RC.OK → result will contain error description ({string}).
  • If code is WebcamScreenshot.RC.OK:
    • If the postTo option has been specified → result will contain the response from the server.
    • If the postTo optionw wasn't set → result will be undefined.
